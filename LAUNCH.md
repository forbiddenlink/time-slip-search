## Algolia Challenge Submission

### Project Title
TimeSlipSearch - Cultural Time Machine

### Short Description
A conversational search interface that lets users explore any date in history (1958-2020) through Billboard charts, movies, prices, and historic events.

### What Makes It Unique
1. **Natural language date parsing** — "Summer of '69" and "Christmas 1985" just work
2. **Multi-index search** — Queries 4 Algolia indices in parallel for instant results
3. **Immersive retro UI** — VHS/CRT aesthetic creates emotional connection to the past
4. **Rich data** — 420K+ records across music, movies, prices, and events

### Tech Stack
- Next.js 14 (App Router)
- Algolia Search (4 indices)
- TypeScript
- Tailwind CSS
- Vercel deployment

### Algolia Features Used
- Multi-index search with `searchSingleIndex`
- Numeric filters for date ranges
- Parallel query execution with `Promise.all`
