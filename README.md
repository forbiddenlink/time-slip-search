## 🔮 What It Does

A searchable database of documented time slip experiences and temporal anomaly accounts from history. Built to explore the intersection of history, paranormal research, and modern search UX.

## ✨ Features

- 🔍 Full-text search across time slip accounts
- 📅 Filter by era, location, and type
- 📖 Detailed account pages
- 🗂️ Categorized archive browser

## 🚀 Getting Started

```bash
git clone https://github.com/forbiddenlink/time-slip-search
cd time-slip-search
npm install
npm run dev
```

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
